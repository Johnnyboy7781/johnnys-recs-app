How to use

Filtering

Use the filters on the left side to search for recommendations in your area.

You must select a region to begin!

Stars

Stars are a custom rating system. Entries can have 0, 1, or 2 stars, and the meaning of a star value is explained below. Despite the number of stars a place may have, please note that every place that exists in this recommendations list is worth your time!

0 - This place is great and excels in a few aspects.

1 - This place is excellent and excels in most aspects!

2 - This place is a MUST-SEE and excels in nearly every aspect!

Superlatives

Superlatives denote that a place is the absolute best at something in a given region. Only one place in a region may have a certain superlative. Places with a superlative will have a special label.
